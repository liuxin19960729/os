# 进程
```
程序计数器(PC):程序当前执行指令单位置
栈指针(SP):
    栈帧用来管理
        函数的参数栈
        函数内部的变量
        函数返回地址


程序保存在磁盘(disk)里
程序运行:
    1.给运行时栈分配内存
    2.os也可能给程序分配一些堆内存(起初堆的大小很小)
        note:c程序中malloc 分配堆内存 free释放堆内存
    3.在运行程序前需要做一下初始化的任务
        例如:
        Unix中回打开stdin stdout stderr 文件描述
    4.os将程序的程序和静态数据加载到内存
    5.启动程序
        OS讲CPU的控制转交给新创建的进程


进程的状态(简单来讲)
    running:正在CPU上执行者指令
    ready:进程准备好运行,OS此时并没有调度该进程
    blocked:执行某个操作,知道其他事件该进程从阻塞状态变为准备状态
```
